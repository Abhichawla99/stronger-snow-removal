## SEO Features Implemented

### 1. Sitemap (`/app/sitemap.ts`)
- Automatically generated XML sitemap
- Includes all pages with priorities and change frequencies
- Accessible at `/sitemap.xml`

### 2. Robots.txt (`/app/robots.ts`)
- Allows all search engines
- Points to sitemap
- Blocks admin/api routes

### 3. Meta Tags
- Title tags optimized for each page
- Meta descriptions (150-160 characters)
- Open Graph tags for social sharing
- Twitter Card tags
- Keywords (where appropriate)

### 4. Structured Data (JSON-LD)
- LocalBusiness schema on homepage
- Business information, ratings, contact details
- Geographic coordinates for Red Deer

### 5. Internal Linking
- Pages link to each other where relevant
- All pages link back to homepage
- Location pages cross-link
- Service pages link to pricing

### 6. Content Optimization
- H1 tags with target keywords
- H2/H3 tags for content structure
- Natural keyword usage
- Local SEO focus (Red Deer, Alberta)
- Long-tail keyword targeting
